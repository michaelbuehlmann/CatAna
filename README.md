# Requirements
- C++11 compatible compiler (GCC for OpenMP)
- Boost library
- FFTW3 library
- HEALPix library (C++)
- GSL library

# Installation
- `git clone` the repository
- run `git submodule init` and `git submodule update`
- create build folder, within build, run `cmake {PATH TO REPO}`
